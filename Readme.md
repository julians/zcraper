Scrapes ZON

# Installation

1. `python3 -m venv venv`
2. `source venv/bin/activate`
3. `pip install -r requirements.txt`

# Run

1. `source venv/bin/activate`
2. `python scrape.py`

# To generate stats about scraped aufmachers

1. `source venv/bin/activate`
2. `python download_xml.py`
3. `python make_stats.py`
